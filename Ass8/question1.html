CSS Pseudo-Classes Demonstration

:hover
Changes style when the mouse cursor hovers over an element. Commonly used for links and interactive elements.
Hover over me!
Move your mouse over the link
:focus
Applies styles when an element receives focus, typically through keyboard navigation or mouse clicks (for form inputs).
Click inside the input or press Tab
:active
Styles an element when it is being activated by the user (e.g., when a button is being clicked).
Click and hold the button
:nth-child()
Selects elements based on their position among siblings. Useful for styling alternating rows, specific positions, or patterns.

  • Item 2 (even)
