Box-Sizing Comparison

Understanding the difference between default and border-box sizing

Default Box-Sizing (content-box)

With default box-sizing, width and padding are added together. The total width = width + padding + border.

Width: 300px

Padding: 30px

Border: 5px

Total Width: 370px

Border-Box

With box-sizing: border-box, padding and border are included in the width. The total width = specified width.

Width: 300px

Padding: 30px

Border: 5px

Total Width: 300px

Key Differences

Content-Box (Default)

  • Width/Height = content only
  • Padding adds to total size
  • Border adds to total size
  • More predictable sizing

Border-Box

  • Width/Height = content + padding + border
  • Padding included in width
  • Border included in width
  • Easier responsive design

Both boxes have the same CSS width, padding, and border values
