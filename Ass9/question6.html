Responsive CSS Cheat Sheet

Quick reference guide for responsive design concepts

CSS Filters

Apply visual effects to images and elements

filter: blur(5px);
filter: grayscale(100%);
filter: brightness(150%);

Media Queries

Apply styles based on screen size

@media (max-width: 768px) {
  .element { display: block; }
}

Flexbox Layout

One-dimensional flexible box layout

display: flex;
justify-content: center;
align-items: center;

Grid Layout

Two-dimensional grid-based layout

display: grid;
grid-template-columns: repeat(3, 1fr);
gap: 1rem;

Responsive Typography

Fluid font sizing with clamp()

font-size: clamp(1rem, 2vw, 2rem);
line-height: 1.6;

Viewport Units

Sizing relative to viewport dimensions

width: 100vw;
height: 100vh;
font-size: 5vw;

Responsive Images

Flexible images that scale properly

img {
  max-width: 100%;
  height: auto;
}

Common Breakpoints

Standard screen size breakpoints

Mobile: 480px
Tablet: 768px
Desktop: 1024px
